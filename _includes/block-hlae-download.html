System requirements:

- Microsoft Windows XP / Vista / 7 or similar
- Microsoft .NET Framework 4.0 or newer
- Genuine Steam version
- OpenGL 1.1 or newer
- a lot of patience
 

Tip:

Using Steam's offline mode you can avoid retriving updates that break HLAE during movie production.
